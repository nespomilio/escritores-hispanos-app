Formatear manuscrito para KDP

Pega tu manuscrito completo (usa "Capítulo" o "###" al inicio de una línea para marcar el comienzo de cada capítulo — empezará en página nueva).

El PDF (vista previa e imprimir) respeta el margen interior/exterior espejado real, página por página, tal como lo pide KDP. El Word usa el margen interior como margen izquierdo fijo en todas las páginas — funciona para revisar y editar, pero antes de subirlo a KDP, en Word activa Diseño de página → Márgenes → Márgenes personalizados → "Páginas opuestas" para que el espejado quede perfecto, o usa directamente el PDF generado aquí.

¿Necesitas EPUB para Kindle? Genera primero tu Word aquí, y conviértelo gratis con Reedsy Book Editor o Calibre — el EPUB es un formato "líquido" (sin páginas fijas), por lo que no se beneficia de las medidas exactas que sí necesita el PDF/tapa blanda.
